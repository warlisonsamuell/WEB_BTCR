MOTO ACADEMY 3.0

Este projeto é parte integrante do curso Motoacademy WEB Fullstack, sediado no Instituto Eldorado de Pesquisa em Manaus. Criado com o objetivo de colocar em prática todo o conhecimento adquirido pelos quatro integrantes do curso de WEB Fullstack, o Boto Cor de Rosa é mais do que uma plataforma de cadastro de currículos. É o resultado do esforço e dedicação de uma equipe comprometida em desenvolver habilidades e oferecer soluções inovadoras no mundo digital. Junte-se a nós nessa jornada e faça parte dessa experiência única de aprendizado e crescimento profissional.
